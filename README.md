
# Tug of War Maths Game 🎮

## Overview
A live, multiplayer maths tug-of-war game with:
- React frontend
- Node.js + Express backend
- Socket.IO for real-time leaderboard

## Project Structure
tug-of-war-maths/
├── frontend/
├── backend/
└── README.md

## Setup

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start

## Features
- Teacher dashboard
- Global leaderboard
- Real-time updates
